# Civilization Builder v3

A turn-based nation-management web game built with React + Vite.

## Deploy on Netlify

Build command:

```bash
npm run build
```

Publish directory:

```txt
dist
```

Upload to GitHub:

- index.html
- package.json
- README.md
- netlify.toml
- .nvmrc if visible
- src/

Do not upload:

- node_modules/
- dist/
- package-lock.json
- the ZIP file itself
