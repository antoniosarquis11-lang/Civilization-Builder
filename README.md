# Civilization Builder v2

A turn-based nation-management browser game built with React/Vite.

## Version 2 upgrades

- Year-by-year gameplay loop instead of instant simulation
- Annual budget allocation and political capital
- Interactive national crises and policy decisions
- Factions with approval levels
- Strategic projects that take several years
- Elections for democracies and regime-survival pressure for authoritarian systems
- Generated flag and visual national identity
- Living nation panel with skyline, mood, environment, and development visuals
- Event illustrations and smoother animated UI
- Local browser save system

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
