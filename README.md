# Civilization Builder v4

A browser-based nation management game. Create a fictional country, choose its population, regime, resources, taxes and policies, then govern through 50 years of development, crises, factions, projects and regional diplomacy.

## Local run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify settings

Build command:

```bash
npm run build
```

Publish directory:

```txt
dist
```

## Upload to GitHub

Upload:

```txt
index.html
package.json
README.md
netlify.toml
.nvmrc
src/
```

Do not upload:

```txt
node_modules/
dist/
package-lock.json
civilization-builder-v4.zip
```
